# 👋 About Me — Personal Portfolio

Interactive personal website with a fun easter egg.

**Live Demo**: [https://veterokk8.github.io/About-me/](https://veterokk8.github.io/About-me/)

---

## ✨ Features

- ⚡ Modern stack: **React 19 + TypeScript + Vite**
- 🌗 Light / Dark theme toggle
- 🎮 Fun interactive "Yes/No" game (the "No" button runs away — try to catch it!)
- 🚀 Fast and performant

## 🛠 Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: CSS Modules (ready to migrate to Tailwind)
- **Animations**: CSS + planned Framer Motion
- **Deployment**: GitHub Pages

## 📋 Sections

- Hero section with greeting
- About me
- My qualities
- Projects (including [Resistance](https://github.com/veterokk8/Resistance))
- Interactive Yes/No game

## 🚀 Getting Started

```bash
git clone https://github.com/veterokk8/About-me.git
cd About-me
npm install
npm run dev
